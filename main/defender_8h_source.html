115/*
116 * A Defender topic has the following format:
117 *
118 * <Prefix><Thing Name><Bridge><Report Format><Suffix>
119 *
120 * Where:
121 * <Prefix> = $aws/things/
122 * <Thing Name> = Name of the thing.
123 * <Bridge> = /defender/metrics/
124 * <Report Format> = json or cbor
125 * <Suffix> = /accepted or /rejected or empty
126 *
127 * Examples:
128 * $aws/things/THING_NAME/defender/metrics/json
129 * $aws/things/THING_NAME/defender/metrics/json/accepted
130 * $aws/things/THING_NAME/defender/metrics/json/rejected
131 * $aws/things/THING_NAME/defender/metrics/cbor
132 * $aws/things/THING_NAME/defender/metrics/cbor/accepted
133 * $aws/things/THING_NAME/defender/metrics/json/rejected
134 */
